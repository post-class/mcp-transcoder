# tool 説明
## browser_navigate
インターネットのホームページURLをパラメータとして渡すと、中身をテキストで取得できます。
スクレイピングや検索で使用してください。
検索したいときは、google.com などの検索サイトを開いて、検索ワードを入力してください。

## browser_install
browser_navigate ツール使用時に、ブラウザがインストールされていないエラーになる事があります。
その場合まず browser_install を実行してください。
